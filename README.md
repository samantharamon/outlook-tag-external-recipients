# Sample: Tag external recipients

## Run unified manifest

1. Download the project files to your computer.
1. In the root of your project, run `npm run build`.
1. Run `npm run start:json` to start the localhost server and sideload add-in.
1. Run `npm run stop` to uninstall the add-in and stop the localhost server.

## Run add-in only (XML) manifest
1. Download the project files to your computer.
1. In the root of your project, run `npm run build`.
1. Run `npm start` to start the localhost server and sideload add-in.
1. Run `npm stop` to uninstall the add-in and stop the localhost server.
